# VibeHealth Mascot Prompts

This file contains ready-to-paste prompts based on the current VibeHealth brand, UI style, and product tone.

## 1) Brand & UX Profile

- **Product**: VibeHealth, a bilingual (EN/FR) personal health companion PWA for medical essentials + daily wellness tracking.
- **Audience**: everyday users who want supportive, non-clinical health guidance and motivation.
- **Core emotional goals**: feel safe, encouraged, and consistent with healthy habits.
- **Brand personality**: warm, playful, reassuring, friendly, premium-soft.
- **Tone of voice**: short, positive, coach-like, never harsh.
- **Visual identity**:
  - Primary warm/coral direction: `#ff6b6b`, `#ff8787`, `#ffa07a`, `#ffcc80`
  - Soft accents: mint/lavender/sky pastels
  - Rounded, organic, “Soft Pop” surfaces with gentle shadows
  - Satoshi typography, clean modern UI, no aggressive blur-heavy visuals
- **Mascot role in app**: progress cheerleader, onboarding guide, success/error state companion.
- **Hard constraints**:
  - Must be original, clearly distinct from famous mascots (Duolingo owl, GitHub Octocat, etc.)
  - Must work as clean 2D vector and remain legible at small icon sizes.

## 2) Mascot Concepts (Summary)

1. **Velvet Hopper**
   - Type: rounded rabbit companion
   - Traits: encouraging, expressive, calm, playful
   - Role: primary in-app assistant for onboarding, streaks, and gentle nudges
   - Distinction strategy: unique oval head proportions, soft heart-shaped nose, asymmetric ear tips

2. **Pulse Sprout**
   - Type: abstract bunny-like wellness spirit (rabbit ears + seed/sprout body)
   - Traits: nurturing, optimistic, soothing
   - Role: low-distraction helper for medical and recovery states
   - Distinction strategy: hybrid creature silhouette (not animal-realistic), botanical motif

3. **Carrobit**
   - Type: geometric rabbit bot (soft corners, no hard tech vibe)
   - Traits: smart, gentle, reliable, cheerful
   - Role: progress/status explainer for charts, reminders, and goals
   - Distinction strategy: geometric body blocks, translucent health “core”, non-owl facial layout

## 4) Animation Prompt Pack (Tool-Ready)

Use these prompts in SVG animation generators, Jitter, or Lottie-style workflows.

- **Idle loop (3s seamless)**  
  Animate the mascot with subtle breathing (2–3px vertical motion), tiny ear sway, occasional blink, and very soft body squash/stretch. Keep motion calm, loop seamlessly, and preserve clean 2D vector style.

- **Success burst (0.9s)**  
  Animate from neutral to joyful pose: quick upward bounce, eyes happy-squint, smile widen, one paw raise, two small sparkle pops, then settle to idle. Keep motion playful but controlled.

- **Thinking cue (1.2s)**  
  Animate slight head tilt, one paw toward chin, brief eyebrow raise, micro pause, then return to neutral. Add a tiny floating question mark bubble for 0.4s if supported.

- **Error/oops response (1.0s)**  
  Animate gentle recoil (small downward dip), concerned expression, brief side-to-side head shake (low amplitude), then return to reassuring neutral expression.

- **Onboarding entrance (0.7s)**  
  Fade + rise from 12px below with soft scale from 0.96 to 1.0, ending in idle pose. Avoid excessive bounce.

- **Reduced motion variant**  
  Replace all large transforms with opacity-only transitions and single blink changes; no continuous bouncing loops.
